# CoolImageApp

## Descrição

Aplicação Android que apresenta fotografias profissionais obtidas através da API da Unsplash.
Permite navegar por imagens, ver detalhes e guardar favoritos.

## API Utilizada

- [Unsplash API](https://unsplash.com/developers)
