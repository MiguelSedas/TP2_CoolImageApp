# Features

## Funcionalidades Principais

1. Obter imagens da API da Unsplash
2. Apresentar imagens numa lista com RecyclerView
3. Atualizar a lista (botão ou swipe refresh)
4. Indicador de carregamento (progress bar)
5. Ecrã de detalhes de cada imagem
6. Suporte a favoritos (FIFO, máximo 5 imagens)
7. Cache local de até 50 imagens
8. Acesso offline às imagens em cache e favoritos
9. Tratamento de erros da API
10. Padrão MVVM
