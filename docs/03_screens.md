# Screens

## Ecrã Principal (MainActivity)

Componentes:

- Toolbar com título "CoolImageApp"
- RecyclerView com grelha de imagens
- Botão de refresh
- Indicador de carregamento (ProgressBar)
- Barra de favoritos com acesso direto às 5 imagens favoritas

## Ecrã de Detalhes (ImageDetailsActivity)

Componentes:

- Imagem em tamanho grande
- Título da fotografia
- Nome do fotógrafo
- Descrição (se disponível)
- Botão para adicionar/remover dos favoritos
- Botão de voltar
