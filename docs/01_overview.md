# Overview

## Propósito

O CoolImageApp é uma aplicação Android que permite aos utilizadores explorar
fotografias profissionais de alta qualidade obtidas através da API da Unsplash.

## Utilizadores Alvo

Utilizadores que procuram inspiração visual e pretendem explorar fotografia profissional
diretamente no seu dispositivo Android.

## Como funciona

1. A app obtém uma lista de imagens da API da Unsplash
2. As imagens são apresentadas numa grelha/lista
3. O utilizador pode carregar numa imagem para ver os detalhes
4. O utilizador pode adicionar imagens aos favoritos
5. A app mantém uma cache local para acesso offline
